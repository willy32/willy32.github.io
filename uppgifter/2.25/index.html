JavaScript instruktion

I denna del lär du dig grunderna inom programmering med JavaScript. Jag rekommenderar att du gör övningarna i rätt ordning, dvs. från första till sista.

JavaScript är ett mycket populärt programmeringsspråk som gör det möjligt att skapa interaktiva webbsidor, t.ex. vad som händer då man klickar på knappar, visa interaktiva kartor, 2D/3D spel samt animationer m.m..

  • Till skillnad från HTML och CSS kan JavaScript klassas som ett programmeringsspråk.
  • JavaScript förkortas ofta som JS.
  • JavaScript kallas också för ECMAScript.
  • JavaScript är inte samma som programmeringsspråket Java.

Vad du redan borde veta

  • En allmän förståelse för Internet och World Wide Web (www)
  • Bra kunskaper i HTML och CSS
